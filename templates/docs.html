SPPU Marksheet Analyzer Documentation

Note: This documentation is for developers and system administrators working with the SPPU Marksheet Analyzer.

Project Overview

A comprehensive web application designed to automate the analysis of SPPU academic marksheets, providing detailed insights into student academic performance across various semesters and years.

Technical Architecture

Component Technology Primary Function
Backend Python, Flask Server-side processing and API routing
PDF Processing PyMuPDF Automated marksheet text extraction
Frontend HTML5, CSS3, JavaScript Interactive user interface

Project Layout

marksheet-analyzer/
├── app.py           # Main Flask application
├── templates/       # HTML view templates
├── static/          # Static resources (CSS, JS)
└── modules/         # Processing modules
    ├── fe/          # First Year modules
    └── te/          # Third Year modules

Quick Setup Guide

git clone https://github.com/yourusername/marksheet-analyzer
cd marksheet-analyzer
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py

Key Application Routes

Route Method Description
`/` GET Landing page
`/upload` POST Marksheet upload endpoint
`/analyze/` GET Performance analysis results

File Upload Workflow

  1. User selects academic semester
  2. Upload PDF marksheet
  3. Backend validates document
  4. Generate performance insights

Logging Strategy

  • Configured at DEBUG level
  • Comprehensive error tracking
  • Detailed processing logs

System Extensibility

Easily expand functionality by adding new semester-specific processing modules in the modules directory.
